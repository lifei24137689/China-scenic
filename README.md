# 5A●Beautiful China — Interactive Map of China's Top Scenic Spots

[![GitHub Pages](https://img.shields.io/badge/demo-online-brightgreen)](https://lifei24137689.github.io/China-scenic/)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

An immersive interactive website showcasing **over 300 of China's highest-rated 5A scenic spots** across 31 provinces. Built as a satellite-style exploration experience with stunning photography, travel guides, and an interactive map.

🌐 **Live Demo:** [https://lifei24137689.github.io/China-scenic/](https://lifei24137689.github.io/China-scenic/)

## 🗺 Features

- **Interactive Satellite Map** — Explore China's scenic spots with a Leaflet-powered map featuring custom aircraft cursor
- **300+ Scenic Spot Pages** — Each with immersive hero images, gallery, travel info, and atmospheric design
- **31 Provincial Regions** — Complete coverage across all Chinese provinces, including Taiwan
- **COS CDN** — All images served via Tencent Cloud COS (Hong Kong region) for fast global access
- **Responsive Design** — Works on desktop and mobile browsers
- **Dark, Cinematic Theme** — Immersive dark UI with parallax effects and smooth transitions

## 🚀 Quick Start

This is a static website — no build tools required.

```bash
git clone https://github.com/lifei24137689/China-scenic.git
cd China-scenic
# Open index.html in your browser or serve with any static server
python -m http.server 8000
```

## 🛠 Tech Stack

- **Frontend:** Vanilla HTML/CSS/JS
- **Map:** Leaflet.js with satellite tile layers
- **CDN:** Tencent Cloud COS (Hong Kong)
- **Hosting:** GitHub Pages + Cloudflare Workers
- **Fonts:** Google Fonts (Orbitron, Rajdhani, Noto Serif SC)
